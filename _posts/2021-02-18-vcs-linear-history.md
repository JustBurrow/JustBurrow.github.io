- 애초에 Git은 리눅스 커널 개발에 사용하기 위한 것이다.
  - Git Flow 역시 리눅스 커널 개발에 좋았기 때문에 나오고, 퍼진 것이다.
    - 오픈소스와 배타적 소프트웨어 판매 회사 모두 Git Flow를 채택한 이유 정리하기.
